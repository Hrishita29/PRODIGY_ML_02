# PRODIGY_ML_02

## Customer Segmentation using K-Means Clustering

### Task Objective
Create a K-Means clustering model to group retail store customers based on their purchasing behavior.

### Dataset
Mall Customers Dataset

### Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

### Machine Learning Algorithm
K-Means Clustering

### Features Used
- Annual Income (k$)
- Spending Score (1-100)

### Project Workflow
1. Load the customer dataset
2. Select relevant features
3. Apply K-Means clustering
4. Assign customers to clusters
5. Visualize customer segments using a scatter plot

### Outcome
Successfully segmented customers into 5 distinct groups based on their income and spending patterns. The visualization helps identify different customer categories for targeted marketing strategies.
